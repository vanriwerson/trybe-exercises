Esse projeto foi criado para trabalhar com Web Storage. Ele armazena as preferências de formatação escolhidas pelo usuário no local storage, ou seja, mesmo que você feche essa aba ou a janela de seu navegador, suas escolhas estarão aqui quando você retornar. Claro que você sempre poderá mudá-las ^^. Abaixo temos um texto genérico cujo único objetivo é ocupar espaço na página. A não ser que queira treinar seu latim, pode ignorá-lo. ^^
